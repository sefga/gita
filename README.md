# 📚 Бхагавад-Гита (Интерактивная книга)

Интерактивное веб-приложение для чтения Бхагавад-Гиты в поэтическом переложении с номерами стихов.

## 🌟 Особенности

- 📖 18 глав с удобной навигацией
- 🔢 Стихи отформатированы по номерам для легкого восприятия
- 🌙 Тёмная и светлая темы для комфортного чтения в любое время суток
- 📱 Адаптивный дизайн для мобильных устройств и планшетов
- ⚙️ Настройки размера шрифта, высоты строк и ширины текстового поля
- 💾 Автоматическое сохранение прогресса чтения (запоминает последнюю главу)
- ⌨️ Быстрое переключение глав с помощью клавиш ← / →

## 🚀 Быстрый старт

```bash
# Запустить локально с помощью любого HTTP-сервера, например:
npx live-server
# или
python -m http.server
```

## 🛠 Технологии

- Vanilla HTML/CSS/JS
- [marked.js](https://marked.js.org/) для рендеринга Markdown-разметки
- JSON для асинхронной подгрузки глав
